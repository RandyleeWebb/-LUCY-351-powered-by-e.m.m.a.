# Uploaded legacy resources

These are the original uploaded server resources copied under the FiveM `resources` folder for convenience.

Do **not** blindly `ensure` this whole folder. Several resources still require QBCore, oxmysql, qb-target, and qb-menu.

Standalone boot path:

```cfg
ensure lucy_framework
ensure lucy_bridge
```

Use these legacy files as source material or manually enable only the ones you have dependencies for.
