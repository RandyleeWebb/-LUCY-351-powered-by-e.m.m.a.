# report-assembly.md

## Assembly Pipeline

1. **Load Template**: Open `report_template.md`.
2. **Progressive Drafting**: Do not write the entire report in one block. Generate section by section.
3. **Executive Summary First**: Write the executive summary after the body is completed, but place it at the top.
4. **Citation Hooking**: Ensure every factual claim correlates to an item in the bibliography. Use inline brackets like `[1]`, `[2]`.
5. **Bibliography Generation**: Keep a running array of all URLs/Sources. Dump them all at the end.
6. **File Output**: Final output must be written to `~/Documents/[Topic]_Research_[YYYYMMDD]/[Topic]_Final.md`.
