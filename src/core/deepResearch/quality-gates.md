# quality-gates.md

## Quality Standards
1. **Source Count**: A minimum of 10+ distinct sources must be utilized.
2. **Claim density**: At least 3 citations per major claim/finding.
3. **Immediacy**: All claims must be cited immediately inline. No deferred citations.
4. **No Fabrication**: Placeholders (like `[Citation needed]`) and hallucinated links are STRICTLY FORBIDDEN.
5. **Format standard**: Prose-first (>=80%). Bullet points must be used sparingly, only for dense lists.
